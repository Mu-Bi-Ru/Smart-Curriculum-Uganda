How the Dashboard Helps You

The CurriSmart Dashboard is designed to empower students with real-time academic insights. Whether it's tracking test performance, keeping up with attendance, or checking weekly feedback, this tool keeps everything clear and structured.

Built with simplicity and accessibility in mind, the dashboard is usable on feature phones, smartphones, tablets, and PCs. Performance analytics are visualized in an engaging and mobile-friendly format.
